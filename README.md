# Shop API SDK

## Installation
You chould use [Composer](http://getcomposer.org/).

    ``` json
    {
        "repositories": [
            {
                "type": "git",
                "url": "git@codebasehq.com:antevorte/frontend-api-sdks/shop-sdk.git"
            }
        ],
        "require": {
            "collins/shop-sdk": "dev-feature/DEVCENTER-53"
        }
    }
    ```

## Testing
$ php vendor/phpunit/phpunit/phpunit.php tests
